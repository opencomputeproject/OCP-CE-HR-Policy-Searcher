## Item 1: Path Pattern Blocklists (Crawl-Time Filtering)

### Problem
The domain YAML template already documents `allowed_path_patterns` and `blocked_path_patterns` fields (see `config/domains/_template.yaml:21-26`), and 128 of 275 domains have `allowed_path_patterns` set, but **the crawler never reads or enforces them**. Links are extracted in `_extract_links()` (`src/crawler/async_crawler.py:147-172`) with only same-domain and extension checks. The post-crawl URL filter (`src/analysis/url_filter.py`) catches some junk but only **after** pages are already fetched, wasting the `max_pages_per_domain` budget.

In the HB323 run, 93 of 100 crawled pages were irrelevant (38 session-details, 22 nav, 32 developer pages) because there was no crawl-time path filtering.

**New code** -- add path pattern filtering after the extension check:

```python
import fnmatch  # Add to imports at top of file

def _extract_links(self, html: str, base_url: str, domain: dict) -> list[str]:
    soup = BeautifulSoup(html, "lxml")
    links = []
    base_domain = urlparse(domain["base_url"]).netloc

    # Get domain-level path patterns (compiled once per call is fine;
    # _extract_links is called once per page, not per link)
    allowed_patterns = domain.get("allowed_path_patterns", [])
    blocked_patterns = domain.get("blocked_path_patterns", [])

    for a in soup.find_all("a", href=True):
        full_url = urljoin(base_url, a["href"])
        parsed = urlparse(full_url)

        if parsed.netloc != base_domain:
            continue
        if parsed.scheme not in ("http", "https"):
            continue

        path_lower = parsed.path.lower()
        if any(path_lower.endswith(ext) for ext in self.skip_extensions):
            if not (path_lower.endswith(".exe") and "/cgi-bin/" in path_lower):
                continue

        # Crawl-time path filtering using domain config
        if blocked_patterns:
            if any(fnmatch.fnmatch(path_lower, p.lower()) for p in blocked_patterns):
                continue

        if allowed_patterns:
            if not any(fnmatch.fnmatch(path_lower, p.lower()) for p in allowed_patterns):
                continue

        normalized = f"{parsed.scheme}://{parsed.netloc}{parsed.path}"
        if parsed.query:
            normalized += f"?{parsed.query}"
        links.append(normalized)

    return list(set(links))
```

**Key design decisions:**
- Use `fnmatch` (glob-style matching) not regex -- matches the `*` patterns already in domain YAML files
- `blocked_patterns` checked first (reject known-bad before checking allow list)
- `allowed_patterns` only enforced when present (empty list = allow all)
- Patterns match against `path_lower` only (not query string) for consistency
- Start paths always bypass filtering (they're added to queue directly in `crawl_domain()` at line 82, never go through `_extract_links()`)

## Item 2: Content-Area Link Extraction

### Problem
`_extract_links()` runs `soup.find_all("a", href=True)` on the FULL HTML including `<nav>`, `<header>`, `<footer>`, and global navigation menus. On SPA sites, these nav elements contain dozens of irrelevant links that consume the crawl budget.

The HTML extractor (`src/crawler/extractors/html_extractor.py`) already strips these elements for content extraction but `_extract_links()` runs on raw HTML before content extraction.

### What to Change

**File: `src/crawler/async_crawler.py`**

```python
_NAV_TAGS_FOR_LINK_EXTRACTION = ["nav", "header", "footer"]

def _extract_links(self, html: str, base_url: str, domain: dict) -> list[str]:

        el.decompose()

    links = []
    base_domain = urlparse(domain["base_url"]).netloc
    # ... rest of existing logic including path pattern checks from Item 1 ...
```

**Key design decisions:**
- Only remove `nav`, `header`, `footer` tags -- NOT `aside`, `script`, `style` etc. which don't contain meaningful links
- Also remove `role="navigation"` elements (common on modern SPAs)
- Do NOT reuse `HtmlExtractor` class here -- link extraction needs a lighter touch than content extraction. We want links from `<main>`, `<article>`, sidebars, etc. -- just not global navigation
- This operates on a COPY of the soup (BeautifulSoup parsing creates a new tree), so it doesn't affect the content extraction that happens later on `result.content`

### Expected Impact
In the HB323 run: the 38 session-details pages and 22 nav pages came from global navigation links. Removing nav/header/footer before link extraction would eliminate most of these, leaving budget for actual bill-related pages.

Add method to `KeywordMatcher`:

```python
def url_bonus(self, url: str) -> float:
    """Calculate bonus score from URL patterns.

    Args:
        url: The page URL

    Returns:
        Bonus points to add to keyword score
    """
    bonus = 0.0
    parsed = urlparse(url)
    hostname = parsed.netloc.lower()
    path = parsed.path.lower()
    full = f"{path}?{parsed.query}" if parsed.query else path

    # .gov TLD bonus
    if hostname.endswith(".gov") or hostname.endswith(".gov.uk"):
        bonus += _GOV_TLD_BONUS

    # Bill/legislation path patterns
    for pattern in _BILL_PATH_PATTERNS:
        if re.search(pattern, full, re.IGNORECASE):
            bonus += _BILL_PATH_BONUS
            break  # Only count once

    # Bill number in URL (HB323, SB192, etc.)
    if _BILL_NUMBER_PATTERN.search(full):
        bonus += _BILL_NUMBER_BONUS

    return bonus
```

Modify `is_relevant()` (lines 330-358) and `get_failure_reason()` (lines 360-388):

```python
def is_relevant(
    self, result: KeywordMatchResult, content_length: int = 0,
    url: str = "",
) -> bool:
    min_score = self.thresholds.get("minimum_keyword_score", 5.0)
    min_matches = self.thresholds.get("minimum_matches", 2)

    if effective_score < min_score:
        return False

    if result.unique_matches < min_matches:
        return False

    stricter_result = self.check_stricter_requirements(result, content_length)
    if not stricter_result.passed:
        return False

    return True


def get_failure_reason(
    self, result: KeywordMatchResult, content_length: int = 0,
    url: str = "",
) -> str:
    min_score = self.thresholds.get("minimum_keyword_score", 5.0)
    min_matches = self.thresholds.get("minimum_matches", 2)

    effective_score = result.final_score
    if url:
        effective_score += self.url_bonus(url)

    if effective_score < min_score:
        url_bonus_str = f" (url_bonus=+{self.url_bonus(url):.1f})" if url else ""
        return f"Below min score ({min_score}){url_bonus_str}"

    if result.unique_matches < min_matches:
        return f"Below min matches ({min_matches})"

    stricter_result = self.check_stricter_requirements(result, content_length)
    if not stricter_result.passed:
        return stricter_result.reason

    return ""
```

## Item 5: MutationObserver SPA Wait Strategy

### Problem
`playwright_fetcher.py:51` uses `wait_until="networkidle"` which waits for no network requests for 500ms. On SPA sites that poll analytics endpoints or use WebSocket connections, this either times out (14 developer pages timed out at 30s in the HB323 run) or returns before the React/Vue content has rendered.

### What to Change

**File: `src/crawler/fetchers/playwright_fetcher.py`**

Replace `networkidle` with a two-phase wait:
1. Wait for `domcontentloaded` (fast, just need DOM ready)
2. Wait for DOM stabilization via MutationObserver (content rendered)

```python
async def fetch(self, url: str) -> CrawlResult:
    if not self._context:
        await self.initialize()

    domain = urlparse(url).netloc
    await self._rate_limit(domain)

    start = datetime.utcnow()
    page = await self._context.new_page()

    try:
        response = await page.goto(
            url,
            wait_until="domcontentloaded",  # Changed from "networkidle"
            timeout=self.settings.timeout_seconds * 1000,
        )

        if response is None:
            return CrawlResult(
                url=url,
                status=PageStatus.UNKNOWN_ERROR,
                error_message="No response",
                used_playwright=True,
            )

        if response.status >= 400:
            # ... existing error handling unchanged ...

        # Wait for DOM to stabilize (SPA content rendering)
        try:
            await self._wait_for_dom_stable(page, timeout_ms=10000)
        except Exception:
            pass  # Continue with whatever content we have

        content = await page.content()
        title = await page.title()

        # ... rest unchanged ...


async def _wait_for_dom_stable(self, page, timeout_ms: int = 10000, stable_ms: int = 500):
    """Wait until DOM stops changing, indicating SPA rendering is complete.

    Uses MutationObserver to detect when no DOM mutations occur for
    `stable_ms` milliseconds. Falls back to simple wait on error.

    Args:
        page: Playwright page object
        timeout_ms: Maximum time to wait for stability
        stable_ms: Milliseconds of no mutations to consider stable
    """
    await page.evaluate(f"""
        () => new Promise((resolve, reject) => {{
            const timeout = setTimeout(() => {{
                if (observer) observer.disconnect();
                resolve('timeout');
            }}, {timeout_ms});

            let timer = null;
            const observer = new MutationObserver(() => {{
                clearTimeout(timer);
                timer = setTimeout(() => {{
                    observer.disconnect();
                    clearTimeout(timeout);
                    resolve('stable');
                }}, {stable_ms});
            }});

            observer.observe(document.body || document.documentElement, {{
                childList: true,
                subtree: true,
                characterData: true,
            }});

            // Start the stable timer immediately in case DOM is already stable
            timer = setTimeout(() => {{
                observer.disconnect();
                clearTimeout(timeout);
                resolve('already-stable');
            }}, {stable_ms});
        }})
    """)
```

**Key design decisions:**
- `domcontentloaded` + MutationObserver is faster than `networkidle` for most sites
- The 500ms stability window is enough for React/Vue rendering but short enough to not waste time
- The 10s timeout prevents infinite waits on constantly-mutating DOMs
- Wrapped in try/except so if the JS evaluation fails, we still get the page content
- This does NOT break non-SPA sites -- static pages will resolve "already-stable" almost immediately

## Item 8: Auto-Generate Domain YAML from URL(s)

### Problem
Adding new domains requires manually creating YAML with the right structure, fields, ID format, and region tags. Users should be able to give the tool a URL and have it generate a draft domain YAML entry.

### What to Change

**New file: `src/tools/domain_generator.py`**

Create a tool that:
1. Takes one or more URLs
2. Fetches each URL (HTTP first, Playwright fallback)
3. Extracts: page title, domain name, language, whether JS is required
4. Generates a domain YAML entry with sensible defaults
5. Determines the output file based on the URL's country (`.gov` TLD analysis, or ask user)

```python
"""Generate domain YAML configuration from URLs."""

import re
from dataclasses import dataclass
from pathlib import Path
from urllib.parse import urlparse
from typing import Optional

import yaml


@dataclass
class GeneratedDomain:
    """A generated domain configuration."""
    name: str
    id: str
    base_url: str
    start_paths: list[str]
    language: str
    requires_playwright: bool
    region: list[str]
    category: str
    suggested_file: str
    notes: str


def generate_domain_id(hostname: str, path: str = "") -> str:
    """Generate a domain ID from hostname.

    Examples:
        lis.virginia.gov -> va_legislature
        energy.virginia.gov -> va_energy
        www.energy.gov -> us_doe
    """
    # Strip www.
    if hostname.startswith("www."):
        hostname = hostname[4:]

    # Handle US state .gov domains
    parts = hostname.split(".")
    if parts[-1] == "gov" and len(parts) >= 2:
        # state.gov pattern: agency.state.gov
        if len(parts) == 3:
            state = parts[1]  # e.g., "virginia" from "lis.virginia.gov"
            agency = parts[0]  # e.g., "lis"
            return f"{_state_abbrev(state)}_{agency}"
        elif len(parts) == 2:
            # federal: energy.gov, epa.gov
            return f"us_{parts[0]}"

    # Generic: use hostname with dots replaced
    clean = hostname.replace(".", "_").replace("-", "_")
    return clean[:30]


def _state_abbrev(state_name: str) -> str:
    """Convert state name to abbreviation."""
    # Subset of states - extend as needed
    STATES = {
        "virginia": "va", "california": "ca", "texas": "tx",
        "oregon": "or", "new-york": "ny", "massachusetts": "ma",
        # ... etc
    }
    return STATES.get(state_name.lower(), state_name[:2])


def detect_region(hostname: str) -> list[str]:
    """Detect geographic region from hostname."""
    if hostname.endswith(".gov"):
        # US federal or state
        parts = hostname.split(".")
        if len(parts) == 3:
            return ["us", "us_states"]
        return ["us"]
    elif hostname.endswith(".gov.uk"):
        return ["uk"]
    elif hostname.endswith(".europa.eu"):
        return ["eu"]
    # ... extend for other TLDs
    return []


def suggest_output_file(hostname: str) -> str:
    """Suggest which YAML file to add this domain to."""
    if hostname.endswith(".gov"):
        parts = hostname.split(".")
        if len(parts) == 3:
            state = parts[1].lower()
            return f"us/{state}.yaml"
        return "us/us_federal.yaml"
    elif hostname.endswith(".gov.uk"):
        return "uk.yaml"
    # ... extend
    return "new_domains.yaml"


def generate_yaml(domain: GeneratedDomain) -> str:
    """Generate YAML string for a domain entry."""
    entry = {
        "name": domain.name,
        "id": domain.id,
        "enabled": True,
        "base_url": domain.base_url,
        "start_paths": domain.start_paths,
        "max_depth": 2,
        "language": domain.language,
        "region": domain.region,
        "requires_playwright": domain.requires_playwright,
        "rate_limit_seconds": 2.0,
        "category": domain.category,
        "tags": [],
        "policy_types": [],
        "verified_by": "auto-generated",
        "verified_date": "",  # Will be filled
        "notes": domain.notes,
    }
    return yaml.dump({"domains": [entry]}, default_flow_style=False,
                      allow_unicode=True, sort_keys=False)
```

```python
    """Generate domain YAML from URL(s)."""
    from .tools.domain_generator import (
        generate_domain_id, detect_region, suggest_output_file,
        GeneratedDomain, generate_yaml
    )
    from .crawler.fetchers.http_fetcher import HttpFetcher
    from .crawler.fetchers.playwright_fetcher import PlaywrightFetcher
    from .crawler.extractors.html_extractor import HtmlExtractor, load_extraction_config
    from .crawler.detection.js_required import detect_js_required

    settings, _, _ = load_settings()
    http_fetcher = HttpFetcher(settings.crawl)
    await http_fetcher.initialize()
    extractor = HtmlExtractor(load_extraction_config())

    try:

            parsed = urlparse(url)
            hostname = parsed.netloc

            # Fetch the page
            result = await http_fetcher.fetch(url)
            needs_playwright = False

            if result.is_success:
                extracted = extractor.extract(result.content, url)
                needs_js, _ = detect_js_required(result.content, extracted.text)
                if needs_js:
                    needs_playwright = True
            else:
                needs_playwright = True

            if needs_playwright:
                pw = PlaywrightFetcher(settings.crawl)
                await pw.initialize()
                result = await pw.fetch(url)
                await pw.close()
                if result.is_success:
                    extracted = extractor.extract(result.content, url)

            domain = GeneratedDomain(
                name=extracted.title or hostname,
                id=generate_domain_id(hostname, parsed.path),
                base_url=f"{parsed.scheme}://{hostname}",
                start_paths=[parsed.path or "/"],
                language=extracted.language or "en",
                requires_playwright=needs_playwright,
                region=detect_region(hostname),
                category="",  # User should fill in

            if args.dry_run:

                        f.write(f"\n  # Auto-generated from {url}\n")
                        # Write just the domain entry (not the "domains:" wrapper)
                        # ... format appropriately
                else:
                    with open(target_path, "w", encoding="utf-8") as f:
                        f.write(yaml_output)

                print(f"Added domain '{domain.id}' to {target_path}")

    finally:
        await http_fetcher.close()

    return 0
```

```python
async def _auth_form(self, credential: dict) -> bool:
    """Handle form-based authentication."""
    page = await self._context.new_page()
    try:
        await page.goto(credential["login_url"], wait_until="domcontentloaded")
        await page.fill(credential["username_field"], credential["username"])
        await page.fill(credential["password_field"], credential["password"])
        await page.click(credential["submit_button"])
        # Wait for navigation after login
        await page.wait_for_load_state("domcontentloaded", timeout=10000)
        # Cookies are now stored in the browser context
        return True
    except Exception as e:
        print(f"Authentication failed: {e}")
        return False
    finally:
        await page.close()
```
